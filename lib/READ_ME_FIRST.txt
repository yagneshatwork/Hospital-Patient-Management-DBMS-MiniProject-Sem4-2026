Place the MySQL Connector/J JAR file in this folder before compiling.

Download from:
  https://dev.mysql.com/downloads/connector/j/

Steps:
  1. Go to the URL above
  2. Select "Platform Independent"
  3. Download the ZIP file
  4. Extract it
  5. Copy  mysql-connector-j-*.jar  into THIS folder (lib\)

Then run compile.bat
